1. Збір даних

Розширення "Golos" не збирає, не зберігає і не передає ваші голосові записи чи тексти на наші сервери.

Вся обробка голосу відбувається локально або через Web Speech API вашого браузера.

2. Дозволи

  • Microphone: Використовується виключно для розпізнавання мови під час активної сесії.
  • ActiveTab / Scripting: Потрібні для вставки розпізнаного тексту в активне поле на сторінці.
  • Offscreen: Використовується для відтворення звукових сповіщень.
